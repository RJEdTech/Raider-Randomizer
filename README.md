# Raider Randomizer

A complete classroom toolkit for Regis Jesuit High School teachers — name picker, group maker, seating chart, and timer in one tool.

**[Launch Raider Spinner 3 →](https://rjedtech.github.io/Raider-Randomizer/)**

## Selection Modes

- **🎡 Wheel** — Classic spinning wheel with tick sounds and smooth deceleration
- **🎰 Slots** — Slot machine reel that scrolls names vertically
- **🃏 Cards** — Names shuffle face-down and one is revealed
- **🎱 Lottery** — Names bounce as lottery balls; one drops into the chute
- **💡 Spotlight** — Grid of all names with a highlight that sweeps and lands

## Tools

- **Spinner** — Five animation styles for picking a random name. Toggle "Remove after picked" to avoid repeats, with a tracker showing who's already been called.
- **Single Name** — Instant random pick with no animation. Quick and quiet.
- **Lineup** — Randomized ordered list. Great for presentation order.
- **Groups** — Divide students into groups of 2, 3, 4, 5, or 6.
- **Teams** — Split students into exactly 2, 3, 4, 5, or 6 teams.
- **Seating Chart** — Random seating grid with drag-and-drop to rearrange. Adjustable column count (3–8).
- **Timer** — Countdown presets (30s to 5min) or custom entry. Color-coded warnings. Pairs with the spinner.

## Features

- **Saved Class Lists** — Save multiple rosters by name. One tap to load.
- **Sound Effects** — Tick sounds during animations, chime on selection, buzz when timer ends. Mute button for testing days.
- **Fullscreen Mode** — Hides everything except the tool for projecting.
- **Copy to Clipboard** — One-click copy for groups, teams, lineup, and seating chart. Paste into Teams or Canvas.

## Privacy

- No accounts, no login, no ads, no cookies, no tracking
- No data leaves the browser — everything runs locally
- Saved class lists are stored in your browser only
- Nothing is transmitted anywhere, ever

## How to Use

1. Visit the link above and bookmark it
2. Paste student names (one per line or comma-separated)
3. Click **Load Names** — optionally **Save as Class** for next time
4. Pick a tool from the tabs and go
